# Brazil Import Tariff Guide — TEC e Acordos Comerciais

## Sistema de Classificação
- NCM (Nomenclatura Comum do Mercosul): 8 dígitos baseado no Sistema Harmonizado
- TEC (Tarifa Externa Comum): alíquota padrão Mercosul para importações extrazona
- LETEC: Lista de Exceções à TEC — produtos com alíquota diferente

## Impostos na Importação Brasileira
### II (Imposto de Importação)
- Base: valor aduaneiro CIF (Cost + Insurance + Freight)
- Alíquotas típicas por setor:
  - Café verde/cru: 10%
  - Café torrado/solúvel: 16%
  - Soja em grão: 6% (extrazona) / 0% Mercosul
  - Frutas frescas: 10-35% dependendo do NCM
  - Sucos de frutas: 20-35%
  - Produtos processados: 14-35%

### IPI (Imposto sobre Produtos Industrializados)
- Café verde: 0%
- Café torrado: 5%
- Produtos processados: 5-15%
- Frutas frescas: 0%
- Sucos: 10-20%

### PIS/COFINS Importação
- Alíquota combinada: 9,65% (PIS 2,1% + COFINS 7,6%)
- Base: valor aduaneiro + II + IPI
- Aplicável a praticamente todos os produtos

### ICMS (Imposto sobre Circulação de Mercadorias)
- Imposto estadual — varia por estado e produto
- São Paulo: 12% para alimentos básicos, 18% geral
- Calculado por dentro (sobre valor com outros impostos)
- Base efetiva: (valor CIF + II + IPI + PIS/COFINS) / (1 - alíquota ICMS)

## Acordos Comerciais — Reduções de II

### Mercosul (Argentina, Uruguay, Paraguay)
- II = 0% para maioria dos produtos agrícolas
- Form Mercosul obrigatório para benefício
- NCM deve estar na lista positiva do acordo

### ACE 35 — Chile
- Redução progressiva — maioria dos produtos: II = 0% ou muito reduzido
- Café: 0% | Frutas: 0-4% | Soja: 0%

### ACE 59 — Colombia, Ecuador, Venezuela, Peru
- Reduções parciais — varia por NCM
- Café colombiano: redução de ~50% do II
- Frutas: 30-70% de redução dependendo do produto

### ACE 58 — Peru (complementar)
- Similar ao ACE 59 para Peru

### WTO/MFN — EUA, China, EU (sem FTA)
- Alíquotas TEC plenas aplicadas
- Nenhuma preferência tarifária

## Cálculo do Custo Landed
Fórmula simplificada:
1. Valor CIF = Produto + Frete + Seguro
2. II = Valor CIF × alíquota_II
3. IPI = (Valor CIF + II) × alíquota_IPI
4. PIS/COFINS = (Valor CIF + II + IPI) × 9,65%
5. ICMS = (Valor CIF + II + IPI + PIS/COFINS) / (1 - 0,18) × 0,18
6. Custo Total = Valor CIF + II + IPI + PIS/COFINS + ICMS
7. Carga tributária % = (Custo Total - Valor CIF) / Valor CIF × 100

## Regimes Especiais
- Drawback: suspensão de impostos para produtos que serão reexportados
- Recof: regime aduaneiro especial para indústrias
- Admissão Temporária: para produtos que retornam ao exterior
